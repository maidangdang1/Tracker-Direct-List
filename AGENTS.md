你需要完成这样一个小小的开源项目：
1. 使用 github action 每隔1个小时完成以下操作：
    - 从`https://github.com/ngosang/trackerslist/blob/master/trackers_all.txt`获取最新的tracker列表
    - 将这个列表转换成`https://github.com/Aethersailor/Custom_OpenClash_Rules/blob/main/rule/Custom_Direct_Classical.yaml`这样的格式
    - 如果转换后的文件与之前的文件不同，就提交到github仓库
    - 提交时，添加`update trackers list`的commit message，带上修改的时间戳
2. 完成 README，包括中文和英文版本
